Circles

Hack Western II Project

Platform:

Android


Built With:

Java, XML, Android Studio


Start to End Date:

November 2015 - Present


Project Goal:

To develop a infinite, skill-demanding android game that has an original concept while also learning XML and developing on the Android platform.


Description:

Circles is an Android game that was conceived at Hack Western II. The game is a move and dodge game and continues until the player fails to dodge 'circles' of other colours (i.e. the player controlled 'circle' collides with another 'circle' of a different colour). Players move by tilting the Android device and gain points when 'circles' of the same color collide. 'circles' of random colours are ejected from the player controlled 'circle' when the player taps the screen.


Current Work:

Although a working version was submitted to Hack Western, my partner and I are still working to improve game experience and add new game elements, with the goal of publishing the project on Google Play.

Check out the project on: Github and the submitted version on: Devpost

Orbit Game

SE 101 Project

Platform:

Arduino, Tiva C Launchpad, Orbit Booster Pack


Built With:

C, Energia


Start to End Date:

September 2015 - November 2015


Project Task:

To develop a program on the the Launchpad using the Orbit Booster Pack.


Description:

The proposed project is a dodging game that utilises the accerometer of the Launchpad to move a sprite. The image is to be displayed on the small screen of the Launchpad. The goal is to dodge falling blocks as they get progressively faster and greater in number. A shield power-up was implemented so that it can protect the sprite from one falling block for a number of seconds. The switches of the booster pack was used to navigate between menu, game, scoreboard and credits page.

Check out the project on: Github

Land Battle Chess

ICS 4U7 - Internal Assessment

Platform:

Windows


Built With:

Java, Object Oriented Programming Model


Start to End Date:

November 2014 - March 2015


Project Task:

To computerize a 2 player version of the chinese board game "Luzhanqi" (land battle chess) that is fully functional.


Description:

A description of the game can be found here. This project uses Java Swing class to display the board in a frame and accepts console input to select pieces on the board. An added feature of the computerized version is the ability to undo turns until the start of the game, to allow players to develop strategies achieved with the use of stacks implemented with linked lists. Exception handling is also implemented for user input. The whole development procedure was documented (from planning, to timeline, documentation of programming techniques and video presentation), as well as code which was done in Oracle standards including.

Check out the project and its documents on: Github
